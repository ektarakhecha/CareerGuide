## 🚨 Troubleshooting Quick Fix

### Issue: Chatbot not responding
```
1. Open DevTools (F12)
2. Check console for errors
3. Verify API key: console.log(GEMINI_API_KEY)
4. Test fallback: chatbotEngine.processMessageWithFallback("hello")
5. Check your internet connection
```

### Issue: Slow responses
```
1. Try gemini-1.5-flash model (faster)
2. Check internet speed
3. Monitor network tab in DevTools
4. Reduce maxOutputTokens if needed
```

### Issue: "API key invalid"
```
1. Verify key in js/gemini.js line 122
2. Check it's not modified
3. Try refreshing page
4. Clear browser cache
5. Try incognito/private mode
```
